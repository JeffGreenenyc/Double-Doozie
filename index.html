How I work

I start with the business decision and define success like an adult: one primary KPI, a few supporting metrics, and guardrails so we don’t “win” by breaking something else. Then I build the data foundation in SQL, validate it, analyze in Python (and R when it’s the right tool), and operationalize the output in a dashboard and playbook that someone can actually run week to week.
